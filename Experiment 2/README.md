# Experiment – Bootstrap Navbar & Card Components

## Overview
This experiment demonstrates the implementation of **Bootstrap UI components**, specifically the **Navbar** and **Card**, to build responsive and reusable layouts for modern web applications.

The purpose of this experiment is to help students understand how Bootstrap simplifies UI development through predefined classes, responsive utilities, and component-based design.

---

## Components Covered

### 1. Bootstrap Navbar
The **Navbar** component is used to create a responsive navigation header that adapts to different screen sizes.

#### Features
- Responsive layout with collapse support  
- Brand/logo section  
- Navigation links  
- Mobile-friendly toggle button  
- Easily customizable using Bootstrap utility classes  

#### Learning Outcome
Understand how to:
- Integrate Bootstrap via CDN  
- Use responsive navigation patterns  
- Implement collapsible menus for mobile devices  

---

### 2. Bootstrap Card
The **Card** component is a flexible and extensible content container.

#### Features
- Card header, body, and footer  
- Image support  
- Buttons and text content  
- Grid-based responsive layout  

#### Learning Outcome
Learn how to:
- Structure content using cards  
- Combine cards with Bootstrap grid system  
- Build clean and consistent UI sections

## Installation & Setup

1. **Create a new project folder**
```bash
mkdir bootstrap-components
cd bootstrap-components
```

2. **Create `index.html`**
Include Bootstrap CDN:
```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
```

3. **Run the project**
Simply open `index.html` in a browser.

---

## Usage

- Use the **Navbar** to navigate between sections of the page  
- Display information using **Cards** for products, profiles, or articles  
- Modify Bootstrap classes to experiment with colors, layouts, and spacing
