# Experiment 2 Bootstrap and material UI


## Overview

This experiment demonstrates the implementation of **Material UI (MUI)
components** within a **Next.js App Router** application to build a
modern, interactive, and component-driven user interface.

The objective is to familiarize students with industry-standard UI
frameworks and illustrate how component libraries simplify the
development of scalable, responsive, and professional web applications

------------------------------------------------------------------------

## Aim

To design and implement an advanced user interface using **Material UI
components** integrated with a **Next.js** application, focusing on
component-based architecture and interactive UI elements.

------------------------------------------------------------------------

## Components Covered (Material UI)

### 1. AppBar & Toolbar

Used to create a responsive application header for branding and
navigation.

### 2. Drawer

Provides side navigation for secondary menus and dashboards.

### 3. Tabs & Tab

Used to organize content into logical sections.

### 4. Card

Displays grouped content such as controls, status, or feedback panels.

### 5. Slider (Temperature Control)

Allows users to adjust numeric values interactively.

### 6. Switch & Chip

Used for system toggles and status indicators.

### 7. Alert & Snackbar

Provides notifications and system feedback.

### 8. Accordion

Displays expandable and collapsible content sections.

### 9. Dialog

Used to show modal information and confirmations.

### 10. Rating

Collects user feedback through ratings.
